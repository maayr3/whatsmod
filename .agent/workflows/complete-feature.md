// turbo-all
1. Identify the task or feature completed.
2. Run the automation script to commit and push changes.
```bash
bash scripts/auto_commit_push.sh "feat: completed <feature_name>"
```
3. Update the `walkthrough.md` with the changes.
4. Notify the user of the completion.
